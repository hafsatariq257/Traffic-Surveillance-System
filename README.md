TrafficAI Tracker
Intelligent Traffic Enforcement & Real-time Vehicle Analytics
A high-performance Computer Vision framework designed for Real-time Vehicle Analytics, Kinematic Speed Estimation, and Automated Forensic Violation Tracking. This system leverages YOLOv8 to monitor urban traffic flow and enforce road safety protocols through AI-driven insights.

✨ Features
🔍 Multi-Class Perception – Precision detection of Cars, Motorcycles, Buses, Trucks, and Bicycles using YOLOv8.

⚡ Kinematic Speed Estimation – Real-time velocity calculation in km/h for every tracked object.

🚨 Automated Enforcement – Instant flagging for:

Red Light Violations: Cross-referencing vehicle position with signal state.

Speed Limit Breach: Automated detection of vehicles exceeding the user-defined threshold.

📊 Live Intelligence Dashboard – Interactive stats featuring total vehicle throughput and violation counters.

📁 Forensic Data Export – Generate and download comprehensive .xlsx logs for administrative review.

🎨 Premium UI/UX – High-visibility Yellow & Black professional theme with fluid Lottie animations.
